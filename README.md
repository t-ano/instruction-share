### 利用した技術
- Laravel 8　（PHPフレームワーク）
- SQLite　（データベース）
- Tailwind CSS　（CSSフレームワーク）

### 要件
- 1か所から出した製造指示を各部署で確認出来る。
- 指示内容をタブレットで確認出来る。
- 毎日の製造指示を保存できる。

### 機能
- 指示の一覧表示、詳細表示、登録、更新、削除。
- 一覧表示の絞り込み。
